PARA CONFIGURAR O SERVIDOR PELA PRIMEIRA VEZ EM SEU COMPUTADOR, utilize os seguintes comandos
no caminho "CampTruco/server-side/":

    - npm install express;
    - npm install cors;
    - npm install mysql2;

Após isso aparecerá uma pasta com nome de NODE_MODULES em seu diretório server-side e um package-lock também!

Agora temos que fazer a conexão com o banco de dados:

Cole o codigo do arquivo dbtruco.sql dentro do mysql. 
Agora o arquivo db.js faz a conexão com o banco.

    Agora é só iniciarmos o codigo. Coloque a pasta do Codigo na pasta C do computador.
    Depois coloque no terminal:

    -cd C:\CampTruco-Proz-main\server-side 
    - node server.js

Pronto agora é só entrar no client-side e abrir as paginas!
